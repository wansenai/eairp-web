## Install and use

- Get the project code

```bash
git clone https://github.com/eairp/eairp.git
```

- Installation dependencies

```bash
npm install -g pnpm

cd web

pnpm install

```

- run

```bash
pnpm serve
```

- build

```bash
pnpm build
```
